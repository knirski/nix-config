# Security policy

## Reporting a vulnerability

Report vulnerabilities through GitHub's
[private vulnerability reporting form](https://github.com/knirski/nix-config/security/advisories/new).

Do not disclose credentials, recovery material, unpublished vulnerabilities,
or sensitive operational details in a public issue.

## Supported version

This repository is continuously maintained. Only the current `main` branch is
supported. Historical plans and revisions remain as learning and audit context,
not as supported releases.

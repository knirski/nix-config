# nixos
- When persisting user home directories via preservation module, use broad directory paths (e.g., `.config` for the entire directory) rather than specifying individual subpaths like `.config/gh`, `.config/git`. Confidence: 0.65
- When fixing display/compositor issues (e.g., suspend/resume with NVIDIA + COSMIC), prefer wayland-native or wayland-idiomatic approaches (e.g., triggering udev `change` events) over killing the compositor or VT switching hacks. Confidence: 0.78
- When powertop's `--auto-tune` re-enables USB autosuspend and overrides udev `power/control=on` rules, prefer `usbcore.quirks` kernel parameters (e.g., `046d:c52b:b` flag) to disable USB autosuspend at the kernel level before any userspace runs, rather than relying on udev rules or powertop exclusion lists — the kernel quirk is immutable and immune to powertop re-enabling autosuspend. Remove any redundant udev rules once the quirk is confirmed working. Confidence: 0.85
- When a stronger, immutable fix (e.g., `usbcore.quirks`) makes earlier fallback code (e.g., udev `power/control=on` rules) redundant or ineffective, remove the dead fallback code rather than keeping it as "belt-and-suspenders." Dead code that doesn't work is dead, not defensive. Confidence: 0.65
- Persist `~/.commandcode` directory via the preservation module's per-user directories list to ensure AI coding assistant state (learnings, taste) survives ephemeral root filesystem resets. Confidence: 0.70
- When a package is a user-facing tool (not system infrastructure), prefer installing it via Home Manager (`home.packages`) rather than NixOS system-wide (`environment.systemPackages`); user-level packages belong in the user profile. Confidence: 0.75
- When systemd-suspend fails with `Failed to freeze unit 'user.slice': Connection timed out` (a known systemd v256+ bug with cgroup freezing), apply `systemd.services.systemd-suspend.environment.SYSTEMD_SLEEP_FREEZE_USER_SESSIONS = false` to skip the user session freeze step. This is especially needed on NVIDIA hybrid-GPU systems where the 60s freeze timeout disrupts suspend/resume timing. Confidence: 0.70
- When injecting environment variables from agenix secrets (e.g., `GITHUB_TOKEN`), prefer Home Manager `programs.bash.initExtra` (user-scoped) over NixOS `environment.extraInit` (system-wide); user-scoped secrets and credentials belong in the user profile alongside user-facing packages. Confidence: 0.70
